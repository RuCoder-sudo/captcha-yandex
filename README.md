##WP Captcha Yandex — Яндекс-капча для WordPress

**WP Captcha Yandex** — это лёгкий и эффективный плагин для WordPress, который защищает ваш сайт от спама и автоматических атак с помощью **Yandex SmartCaptcha**.

Плагин интегрируется с популярными формами и блокирует отправку заявок ботами, не ухудшая пользовательский опыт.

---

### 🚀 Основные возможности

#### 🛡 Защита от спама
- Интеграция с **Yandex SmartCaptcha**
- Надёжная защита форм от ботов
- Минимальное влияние на UX

---

### 🔌 Поддержка форм

Плагин работает с:

- Contact Form 7  
- Elementor Forms  
- WooCommerce  
- Стандартные формы WordPress  
- Любые формы через `wp_mail`

👉 Подключение без сложной настройки
